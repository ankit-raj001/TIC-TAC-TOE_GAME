# Tic Tac Toe (C++)
A simple console-based Tic Tac Toe game developed in C++. This project allows two players to play on a 3×3 board with turn-based gameplay. It was built to practice core C++ programming concepts and game logic.

## Features
- Two-player gameplay
- 3×3 game board
- Win and draw detection
- Input validation
- Console-based interface

## Technologies Used
- C++
- Standard Library

## How to Run

```bash
g++ TicTacToe.cpp -o TicTacToe
./TicTacToe
